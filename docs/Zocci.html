new Zocci(path, optsopt)

The Zocci object which synchronizes a JSON object to disk.
Parameters:
Name Type Attributes Description
path string The filesystem path where the JSON data will be stored
opts object | string | number <optional>
 Options to be used by the Zocci object. If string, acts as `opts.delimiter`. If number, acts as `opts.jsonSpaces`.
Properties
Name Type Attributes Default Description
jsonSpaces number <optional>
 2 The number of spaces used to indent the generated JSON.
delimiter string <optional>
 '.' The symbol used to denote nested objects.

returnKeys boolean If true, returns a list of matching keys. If false, returns the values associated with those keys.
